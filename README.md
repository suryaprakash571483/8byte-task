# 8byte Task

This repository contains the solution for the **8byte assessment task**.  
It demonstrates a frontend application built with React, styled components, and modern JavaScript practices.

---

## 🚀 Features
- Responsive navigation bar
- Hero section with call‑to‑action buttons
- Grid‑based layout for content
- Footer with branding
- Beginner‑friendly CSS styling

---

## 🛠️ Tech Stack
- **React.js** – Component‑based UI library
- **JavaScript (ES6+)** – Logic and interactivity
- **CSS3** – Styling and layout
- **Node.js & npm** – Package management and scripts

---

## 📦 Libraries Used
Dependencies are listed in `package.json`. Common ones include:
- `react` – Core React library
- `react-dom` – DOM rendering
- `react-scripts` – CRA build scripts
- Any additional libraries you installed (e.g., `axios`, `bootstrap`, `tailwindcss`)
